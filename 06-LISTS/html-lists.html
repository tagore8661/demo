HTML Lists

Introduction to Lists

HTML provides three types of lists to organize and structure content:

  • Unordered Lists (<ul>) - For items without a specific order
  • Ordered Lists (<ol>) - For items in a specific sequence
  • Description Lists (<dl>) - For term-definition pairs

Understanding Hypertext in Lists

What is Hypertext?

Hypertext is text that contains links (hyperlinks) to other texts or resources. It allows for non-linear reading and navigation, which is fundamental to how the web works.

List Best Practices

When to Use Each List Type:

Unordered Lists (<ul>)
Use when the order of items doesn't matter
Examples: Features, benefits, ingredients, team members
Ordered Lists (<ol>)
Use when the sequence or ranking is important
Examples: Instructions, rankings, chronological events, steps in a process
Description Lists (<dl>)
Use for term-definition pairs
Examples: Glossaries, metadata, FAQ sections, product specifications

Accessibility Tips:

  • Use semantic list elements instead of styled divs
  • Keep list items concise and scannable
  • Use proper nesting for hierarchical information
  • Ensure sufficient color contrast for list markers
  • Test with screen readers when possible
