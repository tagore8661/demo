HTML Text Formatting Tags

Semantic Formatting Tags

Strong Importance

The strong tag indicates text of strong importance. Screen readers will emphasize this text.

Example: Warning: This action cannot be undone!

Emphasis

The em tag indicates emphasized text with semantic meaning.

Example: I really need to finish this project today.

Mark/Highlight

The mark tag highlights text for reference purposes.

Example: Please review the highlighted sections in the document.

Code

The code tag is used for inline code snippets.

Example: Use the console.log() function to debug your JavaScript.

Keyboard Input

The kbd tag represents keyboard input.

Example: Press Ctrl + C to copy text.

Sample Output

The samp tag represents sample computer output.

Example: The system returned: File not found

Variable

The var tag represents a variable in programming or mathematics.

Example: The formula is y = mx + b

Visual Formatting Tags

Bold Text

The b tag makes text bold without semantic meaning.

Example: The product name is displayed in bold.

Italic Text

The i tag makes text italic for stylistic purposes.

Example: The book title To Kill a Mockingbird is in italics.

Underlined Text

The u tag underlines text (use sparingly as it can be confused with links).

Example: Underlined text for emphasis.

Strikethrough Text

The s tag creates strikethrough text.

Example: Old price $50 New price $30

Small Text

The small tag makes text smaller, often used for fine print.

Example: Terms and conditions apply. See website for details.

Subscript and Superscript

Subscript

The sub tag creates subscript text (below the baseline).

Examples:

  • Chemical formula: H2SO4 (sulfuric acid)
  • Mathematical notation: log2(8) = 3

Superscript

The sup tag creates superscript text (above the baseline).

Examples:

  • Mathematical equation: E = mc2
  • Footnote reference: This statement needs citation1
  • Exponents: 210 = 1024

Quotation Tags

Block Quotation

The blockquote tag is used for longer quotations:

The World Wide Web Consortium (W3C) is an international community where Member organizations, a full-time staff, and the public work together to develop Web standards.

Inline Quotation

The q tag is used for short inline quotations.

As Shakespeare wrote: To be or not to be, that is the question.

Citation

The cite tag is used to reference the title of a work.

I just finished reading The Great Gatsby by F. Scott Fitzgerald.

Abbreviations and Definitions

Abbreviation

The abbr tag defines abbreviations with optional title attribute:

The W3C maintains web standards.

I work for NASA.

Definition

The dfn tag marks the defining instance of a term:

HTML is the standard markup language for creating web pages.

Editorial Tags

Inserted Text

The ins tag marks inserted text:

The meeting is scheduled for January 15th.

Deleted Text

The del tag marks deleted text:

The price is $100 $80 after discount.

Preformatted Text

Preformatted Block

The pre tag preserves whitespace and line breaks:

function greet(name) {
    console.log("Hello, " + name + "!");
}

greet("World");

Contact Information

Address

The address tag defines contact information:

Written by: John Doe
Email: john@example.com
Phone: (555) 123-4567
123 Web Street
HTML City, HC 12345

Time and Date

Time Element

The time tag represents dates and times:

The conference starts on June 15th at 9:00 AM.
