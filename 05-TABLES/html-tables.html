HTML Tables

Basic Table Structure

HTML tables are created using several elements working together:

  • <table> - The container for the entire table
  • <tr> - Table row
  • <td> - Table data cell
  • <th> - Table header cell

Table Accessibility Best Practices

Summary of Best Practices:

  • Always use <caption> to describe the table's purpose
  • Use <th> elements for headers, not <td>
  • Add scope attributes to headers when needed
  • Use <thead>, <tbody>, and <tfoot> for structure
  • Provide alternative text or summaries for complex tables
  • Ensure sufficient color contrast if using colors
  • Test with screen readers when possible

When to Use Tables

✅ Good Uses for Tables:

  • Displaying tabular data (spreadsheet-like information)
  • Comparing data across multiple categories
  • Financial reports, schedules, statistics
  • Any data with clear row/column relationships

❌ Avoid Tables For:

  • Page layout (use CSS Grid or Flexbox instead)
  • Simple lists (use <ul> or <ol>)
  • Positioning elements (use CSS)
  • Creating columns of text (use CSS columns)
